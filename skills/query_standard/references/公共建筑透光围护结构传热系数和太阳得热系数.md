# 表 C.0.1-2 公共建筑透光围护结构传热系数和太阳得热系数的基本要求

来源：GB 55015-2021《建筑节能与可再生能源利用通用规范》附录 C 第 C.0.1 条

适用条件：当公共建筑单一立面的窗墙比大于或等于 0.40 时。

| 气候分区       | 窗墙面积比                   | 外窗传热系数 K [W/(m²·K)] | 综合太阳得热系数 SHGC |
|--------------|------------------------------|--------------------------|----------------------|
| 严寒 A、B 区  | 0.40 < 窗墙面积比 ≤ 0.60     | ≤2.0                     | —                    |
| 严寒 A、B 区  | 窗墙面积比 > 0.60            | ≤1.5                     | —                    |
| 严寒 C 区     | 0.40 < 窗墙面积比 ≤ 0.60     | ≤2.1                     | —                    |
| 严寒 C 区     | 窗墙面积比 > 0.60            | ≤1.7                     | —                    |
| 寒冷地区      | 0.40 < 窗墙面积比 ≤ 0.70     | ≤2.0                     | —                    |
| 寒冷地区      | 窗墙面积比 > 0.70            | ≤1.7                     | —                    |
| 夏热冬冷地区  | 0.40 < 窗墙面积比 ≤ 0.70     | ≤2.2                     | ≤0.40                |
| 夏热冬冷地区  | 窗墙面积比 > 0.70            | ≤2.1                     | —                    |
| 夏热冬暖地区  | 0.40 < 窗墙面积比 ≤ 0.70     | ≤2.5                     | ≤0.35                |
| 夏热冬暖地区  | 窗墙面积比 > 0.70            | ≤2.3                     | —                    |

> 注："—" 表示无限值要求。
