# 表 C.0.6-10 工业建筑的换气次数

来源：GB 55015-2021《建筑节能与可再生能源利用通用规范》附录 C 第 C.0.6 条

| 房间容积 [m³]   | 换气次数 [h⁻¹] |
|----------------|---------------|
| < 500          | 0.70          |
| 501～1000      | 0.60          |
| 1001～1500     | 0.55          |
| 1501～2000     | 0.50          |
| 2001～2500     | 0.42          |
| 2501～3000     | 0.40          |
| > 3000         | 0.35          |

> 注：当房间三面以上外墙有门、窗、暴露面时，表中数值应乘以系数 1.15。
